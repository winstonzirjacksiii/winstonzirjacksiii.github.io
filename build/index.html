What I'm About

I'm a front-end web developer with a penchant for developing clean and usable interfaces. I enjoy and take pride in working with scalable and mantainable stylesheets, building semantic and cross-browser compatable HTML, and peppering content with UX that helps tells stories instead of holding hands.

My experiance is veried. The first leg of my experience in this industry is from a start-up where we designed and built a web-based point-of-sale system. The second was spent at an agency where we built web-apps for tech-industry events. The third leg is currently being filled at an agency where we aid local businesses by designing and implementing sites for popular e-commerce systems.

We all know that it takes four legs to make a large and stable table. Three legs is enough for a small corner table, but my ambitions require the use all of these legs - and then some - to help support and raise a team towards whatever goals they may wish to achieve.

I know that sounds incredibly hyperbolic and a tad bit contrived, but lets be honest here: I just want to use my skills to help people do stuff better. Let us build that fourth leg together so that we can all stand higher and finally replace that proverbial lightbulb left dim above our heads.

  • PNP Suppliments
    View Project

    Shopify, CSS3, Javascript+jQuery, Git, NPM, shopify-themekit, SMACCS, Liquid

    This is one of the first shopify projects in our agency where I was able to develop using the shopify themekit toolkit. Themekit allows for local development, so we were finally able to use NPM build tools and git for organizing code and versioning. We also needed to overcome limits imposed on the shopify system (multiple sections of information attached to products instead of just the single "description" text) while allowing the client to easily mantain the content with full WYSIWYG editing ability. Due to budget constraints, we opted to avoid app development and instead repurposed menu and page objects through a nameing naming system to allow section modularity within a system where it isn't explicitly supported.
