Projects:

Professional:

  • bkr
    View Project

    Shopify, Node.js, Python, CSS3, Javascript+jQuery, Git, NPM, shopify-slate, SMACCS, Liquid

    The plan for this project quickly went from /just/ making small adjustments to a pre-selected theme into a full-scale data migration between their old custom-built system and a Shopify installation. Python and a purpose-built javascript bookmarklet were used for the initial bits of migration, but as the scale changed we built out a custom Node.js script for pulling directly from their MySQL database, joining, and transforming data across multiple tables in order to import a truckload of content into their new environment. Rate limiting was added in conjunction with Shopify's API limitations in order to create a more automated and hands-free migration. Custom UX was written with vanilla javascript, and we reused as much of the existing templating and styling as possible while adding and expanding the existing theme.

  • Socksmith
    View Project

    Shopify, Python, CSS3, Javascript+jQuery, Git, NPM, shopify-slate, SMACCS, Liquid

    This is a recent project in which I was tasked with both migrating to Shopify from Magento alongside building out a custom theme for the project as well. I took a database dump from a Magento installation and with Python and a handfull of libraries converted it into a CSV spreadsheet while transposing and importing information to reflect the changes in environment. On top of that, the theme featured the usual Sleepless Media design and flair, along with a custom-built filter system for the collection view built exclusively with the Liquid templating language.

  • PNP Suppliments
    View Project

    Shopify, CSS3, Javascript+jQuery, Git, NPM, shopify-themekit, SMACCS, Liquid

    This is one of the first Shopify projects in our agency where I was able to develop using the Shopify theme kit toolkit. Themekit allows for local development, so we were finally able to use NPM build tools and git for organizing code and versioning. We also needed to overcome limits imposed on the Shopify system (multiple sections of information attached to products instead of just the single "description" text) while allowing the client to easily maintain the content with full WYSIWYG editing ability. Due to budget constraints, we opted to avoid app development and instead repurposed menu and page objects through a naming system to allow section modularity within a system where those features weren't explicitly supported.
